Описание задачи:
Необходимо сверстать макет страницы по дизайну, предоставленному в Figma. Верстка должна быть выполнена с соблюдением всех современных стандартов и требований. Проект должен быть максимально похож на макет (пиксель-перфект), семантически корректным и оптимизированным для работы на различных устройствах и в разных браузерах.

Требования к верстке:

  1.  Пиксель-перфект:
Верстка должна быть идентична макету в Figma. Максимально точное соблюдение всех отступов, размеров шрифтов, цветов и прочих элементов.
  2.  Кроссбраузерность:
Страница должна корректно отображаться во всех современных браузерах: Chrome, Firefox, Safari, Edge. 
  3.  Адаптивная верстка:
Реализуйте адаптивность для всех экранов: от 320px (мобильные устройства). Проверка должна быть проведена на разных разрешениях экрана. 
  4.  Оптимизация кода и производительности:
  •  По возможности используйте SVG для иконок, оптимизируйте изображения.
  •  Убедитесь в минимальном количестве используемых шрифтов.
  5.  Чистота кода:
Код должен быть чистым, структурированным и легко читаемым. Не допускайте повторений в CSS.
  7.  JS функционал:
- В макете предусмотрены интерактивные элементы: слайдер, выпадающее сверху меню, попап
Слайдеру добавьте анимацию пролистывания и возможность листать его свайпами и по нажатию на кнопки
Меню на мобильном устройстве должно анимированно выпадать сверху по нажатию на бургер и так же возвращаться обратно по нажатию на крестик, появления попапа можно анимировать по своему усмотрению, попап должен закрываться по нажатию на крестик и на темную область экрана вне попапа, внутри попапа текст должен скролится, при этом сам попап остается на месте.
